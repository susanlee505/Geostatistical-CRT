# Geostatistics-CRT
Bayesian Spatial Point Process Method for Cluster Randomized Trial

These are codes that run efficiently on Linux parallel computing.

`libs_and_functions.R` - parameter setting and functions for data simulation and model fitting.\
`run_scenario.R` - parallel computing.

`sensitivity_libs_and_functions.R` - parameter setting and functions for data simulation and model fitting for sensitivity analysis.\
`sensitivity_run_scenario.R` - parallel computing for sensitivity analysis.
